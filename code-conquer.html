About the Event

What is Code & Conquer?

Code & Conquer is a high-intensity dataset analytics challenge for teams who live and breathe data. Participants receive a real-world dataset and must clean, explore, visualise, and build predictive models to uncover hidden insights within a tight time limit.

The event tests your ability to apply statistical thinking, write efficient code, and present findings clearly under pressure. Whether you're a data scientist in training or a machine learning enthusiast, this is your arena to prove your mastery.

Key Rules

  • Teams of 2–4 members.
  • Dataset is provided at the start of the event. No external datasets allowed.
  • Permitted tools: Python (pandas, sklearn, matplotlib, seaborn), Jupyter Notebook, Excel.
  • Internet access is allowed for documentation only — no pre-built model downloads.
  • Submissions must include: cleaned dataset, notebook/code file, and a 3-slide summary.
  • Judging criteria: accuracy, code quality, visualisation clarity, and presentation.
  • Time limit strictly enforced — late submissions will not be accepted.
  • Organiser's decision on scoring is final.
